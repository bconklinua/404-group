# 404-group
Group Project for CMPUT404


## Instructions

### Mac/Linux
```
virtualenv venv --python=python3
source venv/bin/activate
pip install -r requirements.txt
```
### Windows
```
virtualenv venv --python=python3
venv\Scripts\activate
pip install -r requirements.txt
```

### Windows Issues
```
Set-ExecutionPolicy Unrestricted -Scope Process
```
